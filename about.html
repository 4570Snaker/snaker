SMILE RUN is a matching game. Player needs to drag the face to the corresponding choice. There are three levels : Color Matching(Level 1) , Words Matching(Level 2) and Number Matching(Level 3).

Color Matching Mode is to drag the color face to the same color box.

Words Matching Mode is to drag the color face to the same color word.

Number Matching Mode is to drag the number face to the corresponding equation
